# IRC Chatbot for Twitch

This is a chatbot written in clojure based on the Java Kitteh IRC Client Library (KICL)

To start it up, run: 

```
./scripts/run.sh
```

To start a repl: 

```
./scripts/repl.sh
```

# Commands so far

!help
!play <search-for-a-sound-on-freesound.org>
!stop 
!so <username>
Detect when someone joins. 
Detect when someone leaves.
